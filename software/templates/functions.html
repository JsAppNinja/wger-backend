{% trans "What is this?" %}

{% blocktrans %}Workout Manager is a free, open source web application that manages your exercises and workouts. It started as a pet project of mine to replace my growing collection of different spreadsheets, but it has become something other people could also find useful. So, answering the question, it's{% endblocktrans %}

a workout manager

{% blocktrans %}You probably guessed, this is a workout manager: you can create and manage your workout routines. You select how many days your workout constists of, and add different exercises to each.{% endblocktrans %}

{% trans "a body weight tracker" %}

{% blocktrans %}This simple part lets you input your weight for a date. From this it generates a rather nice looking chart, so you can see if you are reaching your goals (be it gaining mass or losing it).{% endblocktrans %}

{% trans "a nutrition manager" %}

{% blocktrans %}Here you can create a diet plan and see some nutritional values (total energy, protein, carbohydrates, and so on) about it. Select from more than 8000 ingredients for this.{% endblocktrans %}

{% trans 'More details please' %}

The workout manager

{% blocktrans %}It is pretty flexible so if you do supersets, no need to worry, it is supported and you will know which exercises belong together. Each set can have a different number of repetitions you want to reach and which you enter yourself. This means that other training forms like pyramid training or similar are not a problem.{% endblocktrans %}

{% blocktrans %}It can also generate a PDF with your current routine. This is suitable for printing so you can bring it to the gym and keep a log about your progress. In case you wonder how it looks like, here's a sample.{% endblocktrans %}

{% blocktrans %}Note: it has no 'preset' workouts that you can simply use (e.g. beginner, etc.). It will happily make the routine you tell it to do, but you have to decide which exercises and how many repetitions you'll do.{% endblocktrans %}

{% trans "Exercise log" %}

{% blocktrans %}You can enter the weights and repetitions you did for each exercise. As with the (body) weight, it will generate some diagrams so so you can see at a glance how well you are doing. The raw numbers are still there in a table, don't worry.{% endblocktrans %}

{% trans "Muscle and exercise database" %}

{% blocktrans %}The exercise database contains around hundred exercises from which you can choose when creating a workout. Some have important notes about the form to perform them. This is displayed in the workout overview page, so you keep them in mind.{% endblocktrans %}

{% trans "The weight tracker" %}

{% blocktrans %}This is a rather simple part that basically lets you input your weight and a day. From this it generates a rather nice looking chart, so you can see if you are reaching your goals (be it gaining mass or losing it).{% endblocktrans %}

{% blocktrans %} In case you want to jump ship after inputing your data from the last years, simply export the data as CSV.{% endblocktrans %}

{% trans "The nutrition schedule manager" %}

{% blocktrans %}And finally it is a nutrition schedule manager. Here you can create a diet plan and see some nutritional values (total energy, protein, carbohydrates, and so on) about it. For the ingredients, it uses the very complete list provided by the US Department of Agriculture, with around 8000 entries.{% endblocktrans %}

{% blocktrans %}It can also be downloaded as a PDF so you can keep it on your fridge, so you don't forget to follow the plan... And as with the workout log, here's a sample.{% endblocktrans %}

{% blocktrans %}Note: this is not a log keeper. It does not keep track of what you actually ate during the day and how it compares to what the plan says. It simply shows you the nutritional values of the plan you create.{% endblocktrans %}
